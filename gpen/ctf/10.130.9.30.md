Hey Jack,

Since this MySQL thing is new to you, I left some notes on how to access the database for you.

We are using docker to run MySQL so you'll have to use docker to access it, but I created a script that will do the majority of the work for you.

Copy and paste the following command to get access to the mysql terminal

docker exec -it mysql sh -c "mysql --user=USERNAME --password=PASSWORD"

Once you are at the mysql> prompt, remember you can run "SHOW DATABASES;"" to see what databases are available.

Reminder that to switch to a new database run the following `use <database_name>;`.

"show tables;" will show you all available tables and lastly `SELECT * FROM <table_name>` will show you all the contents.

I hope these notes help.
